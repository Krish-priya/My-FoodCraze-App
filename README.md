# 🍔 My-FoodCraze-App

A full-stack **Food Delivery Application** inspired by platforms like **Zomato** and **Swiggy**, built using **Java, JDBC, Servlets, JSP, HTML, CSS, JavaScript, and MySQL**. The application provides a seamless food ordering experience with restaurant browsing, menu management, cart functionality, wishlist, and order tracking.

---

## 🚀 Features

### 👤 User Module
- User Registration & Login
- Browse Restaurants
- View Restaurant Menus
- Search Food Items
- Add Items to Cart
- Wishlist Management
- Place Orders
- Order History
- Profile Management

### 🍽️ Restaurant Module
- Restaurant Listing
- Menu Management
- Food Categories
- Item Availability
- Order Management

### 🛒 Cart Module
- Add to Cart
- Update Quantity
- Remove Items
- View Total Price

### ❤️ Wishlist Module
- Add Favorite Food Items
- Remove Items
- Quick Access to Favorite Dishes

### 📦 Order Module
- Place Orders
- Order Confirmation
- Order Status (Open / Closed)
- Order History

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Krish-priya/My-FoodCraze-App.git
```

2. Open the project in Eclipse.

3. Configure Apache Tomcat.

4. Create the MySQL database.

5. Import the SQL script.

6. Update database credentials in the database configuration file.

7. Run the project on Tomcat.
